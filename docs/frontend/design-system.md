# Design Principles

Website harus memberikan kesan:

- Profesional
- Modern
- Cepat
- Bersih
- Terpercaya
- Premium
- Mudah digunakan bahkan saat kondisi darurat (Emergency Friendly)

Prioritas UX:

1. Booking secepat mungkin
2. Informasi mudah dipahami
3. CTA selalu terlihat
4. Mobile First
5. Trust Building
