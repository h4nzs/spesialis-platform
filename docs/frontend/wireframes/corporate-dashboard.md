Version: 1.0

# Goals

Corporate Dashboard merupakan pusat operasional seluruh pelanggan B2B.

Dashboard dirancang untuk perusahaan yang memiliki:

- Banyak cabang
- Banyak order
- Maintenance berkala
- Kontrak layanan
- Invoice bulanan

Tujuan utama:

Memberikan kontrol penuh terhadap seluruh aktivitas maintenance perusahaan.
