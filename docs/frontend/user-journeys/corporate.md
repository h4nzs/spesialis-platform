# Persona

Corporate User merupakan perwakilan perusahaan yang menggunakan layanan B2B.

Contoh:

- General Affair
- Facility Manager
- Building Manager
- Procurement
- Operational Manager
- Office Manager
- Owner

Satu perusahaan dapat memiliki banyak User dengan Role berbeda.

# Goals

Corporate ingin:

- Mengajukan layanan maintenance
- Mengelola banyak cabang
- Memantau seluruh pekerjaan
- Mengelola invoice
- Mengelola kontrak
- Menghubungi Account Manager
- Melihat histori maintenance
